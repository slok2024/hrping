# hrping
hrping是一款免费、轻量级命令行工具。它是 Windows 标准 ping 命令的高级增强版，专门为网络诊断、延迟分析和数据可视化而设计。与普通 ping 不同，hrping 能够提供微秒（$\mu s$）级别的精度，这对于调优低延迟网络环境至关重要。
## 主要核心功能## 
极高的精度： 标准 Windows ping 的精度通常在毫秒级（ms），而 hrping 利用 CPU 的高分辨率计数器，可以达到微秒（$10^{-6}$ 秒）级别的测量精度。
数据统计与可视化： * 它可以计算平均延迟、标准差（Jitter/抖动）以及中位数。支持生成 直方图，直观显示延迟的分布情况。
灵活的发送频率： 你可以设置非常小的时间间隔（如每秒发送 100 次），用于压力测试或捕捉瞬时网络波动。
支持不同的协议： 除了标准的 ICMP 协议，它还支持发送 UDP 包。
无需安装： 这是一个绿色的单执行文件工具，体积非常小，拷贝即用。
